## Current Focus

### 易策 yice

一个零依赖 Python AI 多 Agent 决策参考系统。  
它不是算命工具，而是把周易中的“时、位、变”抽象成结构化分析流程，用六个 Agent 从环境、资源、风险、策略、长期和复盘角度辅助思考复杂决策。

https://github.com/RobbyShao-8ag/yice

### SongStyle AI Design System

一个把宋代美学转译成现代 Web/UI 设计系统的实验项目。  
我希望它不是简单复古，而是把留白、秩序、克制、器物感和东方审美，变成 AI 时代可以执行、可以复用的设计语言。

https://github.com/RobbyShao-8ag/songstyle-ai-design-system
